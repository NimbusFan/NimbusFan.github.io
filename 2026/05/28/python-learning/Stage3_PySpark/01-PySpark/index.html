Python 大数据处理：PySpark RDD 编程基础

Created2026-05-28|Updated2026-05-29|Python 技术笔记大数据与分布式计算
|Word Count:4.8k|Reading Time:17 mins.|Post Views:

基础准备

PySpark的编程模型：

  1. 通过SparkContext对象，完成数据输入
  2. 输入数据后得到RDD对象，对RDD对象进行迭代计算
  3. 最终通过RDD对象的成员方法，完成数据输出工作

PySpark的功能都是从SparkContext对象作为开始

数据输入

RDD对象称之为分布式弹性数据集，是PySpark中数据计算的载体，它可以:

  • 提供数据存储
  • 提供数据计算的各类方法
  • 数据计算的方法，返回值依旧是RDD(RDD迭代计算)
    后续对数据进行各类计算，都是基于RDD对象进行

输入数据到Spark(即得到RDD对象)：

  • 通过SparkContext的parallelize成员方法，将Python数据容器转换为RDD对象
  • 通过SparkContext的textFile成员方法，读取文本文件得到RDD对象

数据计算

map方法

map算子(成员方法)

  • 接受一个处理函数，可用lambda表达式快速编写
  • 对RDD内的元素逐个处理，并返回一个新的RDD
    链式调用
  • 对于返回值是新RDD的算子，可以通过链式调用的方式多次调用算子

flatMap方法

  • 计算逻辑和map一样
  • 比map多一个解除一层嵌套的功能。

reduceByKey方法

  • 接受一个处理函数，对数据进行两两计算

filter方法

  • 接收一个处理函数，可用lambda快速编写
  • 函数对RDD数据逐个处理，得到True的保留至返回值的RDD中

distinct方法

  • 完成对RDD内数据的去重操作

sortBy方法

  • 接收一个处理函数，可用lambda快速编写
  • 函数表示用来决定排序的依据
  • 可以控制升序或降序
  • 全局排序需要设置分区数为1

数据输出

输出为Python对象

  • collect:将RDD内容转换为list
  • reduce:对RDD内容进行自定义聚合
  • take:取出RDD的前N个元素组成list
  • count:统计RDD元素个数

输出到文件中

RDD输出到文件的方法：

  • rdd.saveAsTextFile(路径)
  • 输出的结果是一个文件夹
  • 有几个分区就输出多少个结果文件
    如何修改RDD分区：
  • SparkConf对象设置conf.set(“spark.default.parallelism”, “1”)
  • 创建RDD的时候，sc.parallelize方法传入numSlices参数为1
