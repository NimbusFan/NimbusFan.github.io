For 循环

基础语法

for 循环主要用于遍历（依次取出）数据集中的元素。

这里的“待处理数据集”，严格来说称之为序列类型 。

序列类型是指其内容可以一个个依次取出的一种数据类型，包括：字符串、列表、元组等 。

⚠️ 注意事项：

  • 无法像 while 那样自定义循环条件，只能被动取出数据处理 。

  • 需要通过空格缩进来判断归属 。

变量作用域

for 循环中临时变量的作用域只被限定在循环内。
⚠️ 注意事项：

  • 虽然在 Python 中，你可以在作用域外访问临时变量，但极其不推荐这样做，这破坏了代码的封装性 。

  • 如需访问临时变量，可以预先在循环外定义它

For 循环的嵌套

与 while 类似，for 循环也支持嵌套：

Continue 和 Break

continue

continue 的作用是：中断本次循环，直接进入下一次循环 。

continue 可用于 for 循环和 while 循环，效果一致 。

break

break 的作用更加坚决：直接结束整个循环 。

break 同样可用于 for 循环和 while 循环，效果完全一致 。

💡 Break 的底层作用规则：
在 Python 以及绝大多数编程语言中，break 遵循以下逻辑：

  1. 寻找上级： 当程序运行到 break 时，向外层寻找最近的一个 for/while 循环 。
  2. 无视障碍： 在寻找的过程中，它会穿透所有的 if, elif, else 。
